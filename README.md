## 区块链知识了解和方向
    规划的看个人爱好了。
    对于开发人员来讲，区块链开发大方向分两块：

    第一类是区块链应用开发：如 智能合约 及 DAPP开发， 
    智能合约的开发语言：以太坊平台主要是Solidity语言， 超级账本 是 go 语言，EOS 主要是C++语言， DAPP的话就看应用平台了。

    第二类是公链开发，要求会高一些，最好对分布式系统， 密码学 ， 博弈论 有一些了解。
    公链开发涉及的编程语言主要是go C++  

    当然这两类也会有一些交集，比如为公链开发数字钱包，区块链浏览器等。
